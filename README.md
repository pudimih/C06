## 📂 Estrutura do Projeto

O projeto está organizado por aulas e exercícios.

```text
src/
└── main/
    └── java/
        ├── aula01/
        │   ├── aula/
        │   └── exercicios/
        │
        ├── aula02/
        │   ├── aula/
        │   └── exercicios/
        │
        └── aula03/
            ├── aula/
            └── exercicios/
```

Cada pasta representa:

- **aula** → conteúdo visto em aula + códigos demonstrados pelo professor
- **exercicios** → exercícios resolvidos
